# app-shell

## ADDED Requirements

### Requirement: 首頁橫幅視差捲動
首頁的活動橫幅 SHALL 隨景點列表捲動呈現視差效果——橫幅內容的位移速率 MUST 與列表捲動速率不同（背景移動較慢），效果 MUST 侷限在橫幅顯示區域內，不得使橫幅內容溢出容器或造成版面跳動。

#### Scenario: 捲動列表時橫幅產生視差
- **WHEN** 使用者捲動首頁景點列表
- **THEN** 活動橫幅隨捲動位移，位移速率明顯小於列表本身的捲動速率

#### Scenario: 捲回頂部視差歸零
- **WHEN** 使用者捲回列表最頂端
- **THEN** 橫幅回到未位移的原始位置，無殘留偏移
